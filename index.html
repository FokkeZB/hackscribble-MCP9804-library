Hackscribble MCP9804 - I2C temperature sensor for Arduino.

Comprehensive library for the Microchip Technology MCP9804 digital temperature sensor.

@@@.

@@@.

The library lets you:

  • read ambient temperature with configurable resolution (down to 0.0625 degrees) and rounding
  • set and monitor upper, lower and critical temperature thresholds
  • respond to a hardware alert signal from the MCP9804 when a threshold is exceeded

The library hides all the details of communications, register addressing and value conversions. Just connect the sensor, specify its I2C address and you can start reading temperatures.

Each MB85RS chip adds up to 256KB of fast, non-volatile storage to the Arduino. You can read more about the MCP9804 on the Microchip Technology website.

To test the MCP9804, we designed and built a small evaluation board.
